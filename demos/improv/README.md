# P2D - Improv - learn Powershell via some Improv sessions

The project `P2D` is a github repo for a course on learning
enough DOS/Powershell to be Dangerous(ly) efficient when working
on Windows.


This is a "Improv" session to look at some powershell development
to build some tools to make us more efficient.

We have 2 warm-up exercise with various refinements see folders/projects
